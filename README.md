# All Trails Redesign Assignment

## Description

This React-Native app was produced for my final year unit "Ubiquitous Computing", where the goal was to prototype a redesign/competitor to the [AllTrails](https://www.alltrails.com) app. The prototype was to be designed following the principles of Ubiquitous Computing learnt while undertaking the unit.

## Results

For this assignment I received a 2:1.
